Tokenizer via `tokenizers::Tokenizer::from_file(tokenizer_path)`.

Public method `extract(&self, text: &str) -> Result<Vec<ExtractedEntity>, AppError>`:
1. Call `chunk_text(text, MAX_TOKENS=510, CHUNK_OVERLAP=50, &self.tokenizer)` (per Pitfall 1).
2. For each chunk, call `extract_chunk(chunk)`.
3. Concatenate, sort by (entity_type, value), dedup by (value, entity_type).
4. Return.

Private `extract_chunk(&self, text)`:
1. `self.tokenizer.encode(text, true)` → encoding.
2. Build `Array2<i64>` for input_ids / attention_mask / token_type_ids.
3. Acquire `self.session.lock()`, run `session.run(ort::inputs![...])` per RESEARCH Example 1.
4. Extract logits tensor `(1, seq_len, 9)`, pass to `decode_bio(&logits, encoding.get_offsets(), &self.id2label, text)`.

Private `decode_bio(logits, offsets, id2label, text) -> Vec<ExtractedEntity>`:
1. For each token: argmax over the 9-label dimension, look up the label string.
2. Walk tokens; on `B-*` start an entity span; extend with `I-*` of same type; close on `O` or different `B-*`.
3. For each closed span, use `offsets[start_token].0` and `offsets[end_token].1` to slice the original `text` (per Pitfall 2). This guarantees verbatim surface form even when WordPiece split words.
4. Map labels: `B-PER`/`I-PER` → entity_type `"person"`; `B-ORG`/`I-ORG` → `"organization"`; `B-LOC`/`I-LOC` → `"location"`. Drop MISC and O.
5. Return `ExtractedEntity { label: entity_type.to_string(), value: slice.to_string(), entity_type: entity_type.to_string(), canonical_id: None }` per RESEARCH §Open Questions #1 (drop MISC for v1).

Private `chunk_text(text, max_tokens, overlap, tokenizer) -> Result<Vec<String>, AppError>`:
1. If `tokenizer.encode(text, false).get_ids().len() <= max_tokens`, return `vec![text.to_string()]`.
2. Else: split text into sentences (regex `[.!?]\s+` or naive `.split(". ")`).
3. Greedy pack sentences into chunks ≤ max_tokens by re-encoding each candidate chunk.
4. Maintain `overlap` tokens between consecutive chunks (include last ~50-token suffix of previous chunk at start of next).
