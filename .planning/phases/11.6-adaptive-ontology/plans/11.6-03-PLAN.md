Main public function:

```rust
/// Compute canonical_short_name for one entity per D-10.
///
/// Rules (evaluated in order; first match wins):
///  1. If class=Organization AND corporate suffix present ŌåÆ strip it.
///  2. If class=Location AND value contains a hyphen or space AND last segment
///     matches HYPHEN_UNIT ŌåÆ return last segment.
///  3. Whitespace-only differences ŌåÆ collapse and return normalized form when
///     it differs from `value`.
///  4. Otherwise ŌåÆ None (caller falls back to raw `value`).
///
/// Never returns an empty string. Never returns Some(x) where x == entity.value
/// (idempotent short-name is not a rewrite ŌĆö caller treats None as "no change").

```

```rust
/// Recompute canonical_short_name for a canonical entity as the most-frequent
/// short-name across all its alias occurrences per D-11.
///
/// Behavior:
/// - Gather all doc_ids in doc_index[canonical_id], look up each doc's
///   extracted_entities via engine metadata (SAME lookup pattern already used
///   in recompute_canonical_name; consult existing code for the exact join).
/// - Actually simpler: track the canonical_short_name from ExtractedEntity
///   as they come in via register_doc_entities. Aggregate the mode (most
///   frequent). Ties broken by first-seen order.
/// - When zero aliases have canonical_short_name ŌåÆ set None (falls back to
///   canonical_name in UI per D-11).
///
/// Called at the end of register_doc_entities for every touched canonical_id
/// (parallel to how recompute_canonical_name is called today).
fn recompute_canonical_short_name(
    &mut self,
    canonical_id: &str,
    incoming_short_names: &[Option<String>],
) {
    let counts: HashMap<String, u32> = incoming_short_names.iter()
        .filter_map(|s| s.as_ref())
        .fold(HashMap::new(), |mut m, s| {
            *m.entry(s.clone()).or_insert(0) += 1;
            m
        });

    let most_frequent = counts.into_iter().max_by_key(|(_, c)| *c).map(|(s, _)| s);

    if let Some(c) = self.canonicals.get_mut(canonical_id) {
        c.canonical_short_name = most_frequent;
    }
}
```
