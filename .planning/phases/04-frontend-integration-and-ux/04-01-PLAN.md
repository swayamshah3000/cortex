  - ActivityItem

  ```rust
  #[derive(Debug, Clone, Serialize, Deserialize)]
  #[serde(rename_all = "camelCase")]
  pub struct ActivityItem {
      pub id: String,
      pub action: String,
      pub subject: String,
      pub timestamp: String,
      #[serde(rename = "type")]
      pub activity_type: String,  // "info", "success", "warning", "error"
  ```
