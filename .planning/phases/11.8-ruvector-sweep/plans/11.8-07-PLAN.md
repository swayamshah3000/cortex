<objective>
Add post-cosine attention re-ranking to `search_documents_impl` — but ONLY if 11.8-02-AUDIT.md verdict is GO / GO-WITH-CAVEAT and only when RerankConfig.enabled = true (D-08/D-10/D-11/D-12/D-13). Ships as opt-in experimental toggle (Plan 10 owns the UI); default OFF.

Purpose: HNSW ranks by cosine only. Cosine has known failure modes on semantically-close-but-topically-distinct docs. An attention head trained on relevance produces a stronger secondary signal. Blending both preserves cosine's recall while attention re-orders the top-20 for better precision-at-5.

Output:
- If 11.8-02-AUDIT.md verdict == NO-GO: `11.8-07-DEVIATION.md` written; NO Rust code changed; plan exits.
- If GO: `src-tauri/src/search/attention_reranker.rs` implementing rerank via the selected mechanism; search_documents_impl gains a rerank step gated on RerankConfig; Cargo.toml gets `ruvector-attention`.
- Final score formula becomes `0.5*cosine + 0.3*attention + 0.15*jaccard + 0.05*recency` when enabled; existing formula when disabled.
</objective>
