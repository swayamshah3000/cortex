| local-ruvllm inference → app | User doc content passes through on-device LLM; no network egress |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11.8-06-01 | Tampering | HuggingFace GGUF download | mitigate | validate SHA256 checksum from 11.8-01-AUDIT.md before atomic rename; on mismatch delete .partial and return Err |
| T-11.8-06-02 | Denial of Service | ruvllm init on Metal-unavailable machine | mitigate | wrap init in Result with typed error; ai_request returns `[FALLBACK]`-prefixed error so Plan 10 UI swaps providers |
| T-11.8-06-03 | Information Disclosure | model weights persisted unencrypted | accept | weights are public HF artifacts; user's documents are never persisted alongside them |
| T-11.8-06-SC | Tampering | cargo add ruvllm (path dep) | mitigate | path dep to local `experiments/ruvector` — no npm/crates.io supply chain; audit is 11.8-01 |
